# Laser-Based-Intrusion-Detection-with-Smartphone-Alerts
Overview

A laser-based intrusion detection system using Arduino Uno, LDR, buzzer, and GSM 900A. When the laser beam is interrupted, the system triggers an alarm and sends an SMS/call alert to the owner using a 2G SIM.
